## 🚀 Langkah Selanjutnya

### Untuk Menjalankan Aplikasi:

1. **Setup Backend:**
```bash
cd backend
npm install
# Edit .env file sesuai konfigurasi Anda
npm run dev
```

2. **Setup Frontend:**
```bash
cd frontend
npm install
# Setup React Native environment
npm run android # atau npm run ios
```
